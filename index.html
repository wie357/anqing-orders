⚙️ 首次使用：請設定 Firebase 雲端資料庫

請開啟 HTML 原始碼，將檔案中標示為 FIREBASE_CONFIG 的區塊填入你從 Firebase Console 取得的設定值。

  1. 到 Firebase Console 建立新專案
  2. 新增 Web App（齒輪圖示 → 專案設定 → 應用程式 → 新增 Web 應用程式）
  3. 複製 firebaseConfig 物件中的所有欄位
  4. 到「Firestore Database」→「建立資料庫」→「測試模式」
  5. 把設定值貼回此 HTML 檔的 FIREBASE_CONFIG 區塊
  6. 重新整理本頁

📖 詳細步驟請參考同資料夾中的 部署說明.md 文件。
